# flask_rest_api
Simples API Rest usando Python e framework Flask

# Install
* mkdir flask_api
* cd flask_api
* virtualenv flask
* flask/bin/pip install flask
* flask/bin/pip install flask-httpauth
* chmod a+x app.py
* ./app.py

# Detalhes
*CRUD usando os métodos HTTP GET/POST/PUT/DELETE de um data store de personagens(herois)
*Recursos:
*Autenticação
*Rotas inexistentes
*HTTP STATUS CODE
